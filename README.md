# shell_plotter
A shell plotting function that extends a ruby framework and plots the metagenomics abundances right in your shell for checking the abundance distribution. It takes the OTU metagenomics abundances and then plot the bar and the histogram right in your shell quickly to see the distribution of the abundances.
